1. @for문

- @for 키워드는 스타일을 반복적으로 출력하는데 사용합니다.

- js의 for문과 유사합니다.

    **문법
    through : 종료만큼 반복
    @for $변수 from 시작 through 종료 {
      //반복할 내용
    }

    **문법 2
    //to : 종료 전까지 반복문
    @for $변수 from 시작 to 종료{
      //반복할 내용
    }

$변수 => #{변수}

#{변수}는 코드 어디든지 변수 값을 사용할 수 있습니다.

사용 예) $img: unquote("test01.jpg");
h1{background-image:url('./img/#{$img}');}
